1)gdzie jest wstawiany,

Wstawia się zawsze w części nagłówkowej dokumentu (pomiędzy znacznikami < head> a < /head>).
Można go zastosować, gdy elementy które pragniemy poddać formatowaniu, występują wielokrotnie na stronie i wszystkim chcemy
nadać takie same atrybuty (inne niż domyślne).

2)składnia teoretyczna (w ramce, konieczne wcięcia),

< head>
(...)
< style type="text/css">
/ * < ![CDATA[ */
selektor1 { cecha: wartość; cecha2: wartość2... }
selektor2 { cecha: wartość; cecha2: wartość2... }
( ...)
/ * ]]> */
< /style>
(...)
< /head>















3)jakie mogą być selektory,

Selektorem może być praktycznie dowolny znacznik,
np. p (akapit), h1 (tytuł), td (komórka tabeli) i inne.
To właśnie elementom, które znajdują się w kodzie źródłowym, pomiędzy tymi znacznikami, nadajemy atrybuty formatowania opisane w arkuszu.

4)jakie mogą być cechy i wartości.

Jako "cecha" (w deklaracji stylu - powyżej) należy wpisać o jakie konkretnie atrybuty formatowania nam chodzi (opisane w kolejnych rozdziałach).
Natomiast jako wyraz "wartość" wpisujemy dokładną wartość atrybutu.
