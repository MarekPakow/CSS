Pojęcie Nr identyfikacyjny obiektu,
właściwości lub komunikatu.
Numery właściwości i numery komunikatów
są określane w skrócie jako numery ID.
Co znaczy skrót ID? identifier, czyli identyfikator
Jaki jest cel
stosowania ID?		 przede wszystkim jako sposób linkowania
do konkretnych części dokumentu (kotwica nawigacyjna),
jako uchwyt dla skryptów JS oraz do ostylowania
konkretnego elementu w CSS.
Jakie jest
znaczenie znaku #		 znak # jest używany do oznaczenia selektora identyfikatora .
W CSS identyfikatory są unikalnymi nazwami przypisanymi do elementów HTML,
które można wykorzystać do ich stylizacji.
Jak następuje
wywołanie ID
(wraz z
wytłumaczeniem)?		 Wywołanie ID w CSS polega na użyciu selektora z symbolem #,
który preceduje nazwę identyfikatora przypisanego do elementu HTML.
W ten sposób można stylować pojedynczy element na stronie.
ID musi być unikalne na stronie,
a jego przypisanie odbywa się za pomocą atrybutu id w HTML.
Uwagi na temat
stosowania ID
(trzy) konieczne
wyliczenia.		 1. Unikalność ID: ID w HTML musi być unikalne na całej stronie.
Nie można przypisać tego samego identyfikatora do dwóch różnych elementów,
ponieważ narusza to zasady HTML i może prowadzić do nieprzewidywalnych zachowań w CSS i JavaScript.

2. Specyfika selektora ID: Selektor ID (#id) ma wyższą specyficzność
niż selektory klasowe (.class) czy selektory tagów (element).
Oznacza to, że style przypisane do ID
będą miały priorytet w przypadku konfliktu stylów.

3. Ograniczona elastyczność: ID jest bardziej sztywne i przeznaczone do stosowania w przypadkach,
gdzie elementy są unikalne. Jeśli elementy mają być stylowane
w podobny sposób, lepiej używać klas (.class),
które można przypisać wielu elementom, co zwiększa elastyczność kodu.
